# PesuConnect
## Freelancing website for PESU Students 

### PESUConnect is a full-stack web app that creates a freelance marketplace for university students. Built with a Python Streamlit frontend and a MySQL backend, it allows users to post projects, apply for jobs, and manage their profiles. The database enforces all rules using stored procedures and triggers for a complete workflow, from application to contract, payment, and user reviews.
